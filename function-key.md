# Function Key Behavior (Kinesis Advantage 2)

## Right Hand Fn Layer

When Function key is held, RHS keys produce F-keys:

### Top Row (Standard F-Keys)
- `-` ‚Üí F13
- Hotkey 3 ‚Üí F14
- `Y` ‚Üí F15
- `U` ‚Üí F16
- `I` ‚Üí F17
- `O` ‚Üí F18
- `P` ‚Üí F19

### Middle Row
- `\` ‚Üí F20
- Hotkey 4 ‚Üí F21
- `H` ‚Üí F22
- `J` ‚Üí F23
- `K` ‚Üí F24
- `L` ‚Üí Alt+F1
- `;` ‚Üí Alt+F2

### Bottom Row (Alt Combinations)
- `'` ‚Üí Alt+F3
- `N` ‚Üí Alt+F4
- `M` ‚Üí Alt+F5
- `,` ‚Üí Alt+F6
- `.` ‚Üí Alt+F7
- `/` ‚Üí Alt+F8

### Navigation and Brackets
- `Up` ‚Üí Alt+F9
- `Down` ‚Üí Alt+F10
- `[` ‚Üí Alt+F11
- `]` ‚Üí (unmapped)
