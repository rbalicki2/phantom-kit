# Function Key Behavior (Kinesis Advantage 2)

## Right Hand Fn Layer

When Function key is held, RHS keys produce F-keys:

### Top Row (Standard F-Keys)
- `-` → F12 (config says F13 but lies)
- Hotkey 3 → F14
- `Y` → F15
- `U` → F16
- `I` → F17
- `O` → F18
- `P` → F19

### Middle Row
- `\` → F20
- Hotkey 4 → F21
- `H` → F22
- `J` → F23
- `K` → F24
- `L` → Alt+F1
- `;` → Alt+F2

### Bottom Row (Alt Combinations)
- `'` → Alt+F3
- `N` → Alt+F4
- `M` → Alt+F5
- `,` → Alt+F6
- `.` → Alt+F7
- `/` → Alt+F8

### Numbers (default Kinesis behavior, offset by 1)
- `6` → F7
- `7` → F8
- `8` → F9
- `9` → F10
- `0` → F11

### Navigation and Brackets
- `Up` → Alt+F9
- `Down` → Alt+F10
- `[` → Alt+F11
- `]` → Alt+F12
