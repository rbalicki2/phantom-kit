L-Mode (a from Normal)

Select modifier, then press key:

u = Cmd          Shift = Cmd+Shift
i = Ctrl         Shift = Ctrl+Shift
c = Alt          Shift = Alt+Shift
o = Cmd+Ctrl     Shift = Cmd+Ctrl+Shift
t = Cmd+Alt      Shift = Cmd+Alt+Shift
e = Ctrl+Alt     Shift = Ctrl+Alt+Shift
d = Cmd+Ctrl+Alt Shift = All modifiers

Then press any key to execute modifier+key
Fn+key = modifier+mirrored LHS letter

RCtrl = exit to Normal
Ctrl+t = exit to Ins
