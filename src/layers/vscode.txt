VS Code Layer (right_control+t, VS Code only)

o = Copy relative path, Fn = Copy full path
Ctrl+o = Copy rel path + open in Chrome (cc/isof)
t = Cmd+D (go to definition), Fn = Cmd+. (quick fix)
e = F2 (rename)
a = Cmd+R (find references)
c = Create 2nd tab, Fn = Move to next tab
Fn+i = Move to prev tab
d = Cmd+B (sidebar toggle)
w = Ctrl+` (terminal toggle)
u = Cmd+P paste Enter (open from clipboard)
Fn+u = Copy + Cmd+P paste Enter (open file at cursor)
n = Next error (stays)
r = Find next (stays)
