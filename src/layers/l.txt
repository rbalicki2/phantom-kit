L Layer (a from Normal)

o = plus, Fn = Cmd+plus
h = equals, Fn = Cmd+equals
Ctrl+o = Close (Cmd+W)
Enter = Cmd+Enter

Sub-layers (press, then a-z sends modifier+letter):
u = Cmd, Ctrl = Ctrl+Cmd
i = Ctrl, Ctrl = Ctrl+Alt
c = Alt, Ctrl = Alt+Cmd
d = Cmd+Ctrl+Alt
Fn+key = modifier+mirrored LHS letter
