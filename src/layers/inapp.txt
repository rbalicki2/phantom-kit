In-App Nav (hk4 from Normal or Ins)

J/K = Scroll down/up (hold to repeat), Fn+J/K = End/Home
U/I = Back/Forward (stays)
M/, = Prev/Next tab (stays)
Y = Close tab (stays), Shift+Y = Close + prev tab, Fn+Y = Reopen closed tab
L = Address bar / Go to line -> Ins
O = Open -> Ins
. = Refresh (stays), Fn+. = Super Refresh (stays)
Enter = Cmd+Enter -> Ins

Chrome: H = Cmd+K, Fn+H = Tab search, Shift+H = History
Chrome: N = New tab, Fn+N = New window

VS Code: H = Command palette, Fn+H = Quick open
VS Code: N = New file, Fn+N = New window

iTerm: N = Split pane, Fn+N = New tab

Default: H = Cmd+K, Fn+H = Cmd+P, N = Cmd+N, Fn+N = Cmd+Shift+N

Up = App switcher (J/K to cycle, Enter to select)
Down = Window switcher (J/K to cycle, Enter to select)

right_control = Exit to Normal, Ctrl+J = Exit to Ins
HK4 = Go to Nav layer
