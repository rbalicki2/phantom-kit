In-App Nav (hk4 from Normal or Ins)

t/e = Scroll down/up (hold), Fn = End/Home
i/c = Back/Forward (stays)
s/n = Prev/Next tab (stays)
u = Close tab, Shift = +prev tab, Fn = Reopen
a = Address bar / Go to line -> Ins
d = Open -> Ins
r = Refresh (stays), Fn = Super Refresh
Enter = Cmd+Enter -> Ins

Chrome: o = Cmd+K, Fn = Tab search, Shift = History
Chrome: h = New tab, Fn = New window

VS Code: o = Command palette, Fn = Quick open
VS Code: h = New file, Fn = New window

iTerm: h = Split pane, Fn = New tab

Default: o = Cmd+K, Fn = Cmd+P
Default: h = Cmd+N, Fn = Cmd+Shift+N

right_control = Exit to Normal
HK4 = Go to Nav layer
