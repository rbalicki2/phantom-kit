# RHS Slots for One-Handed Typing

## Modifier Keys Available
- right_command (rcmd)
- right_control (rctrl)
- right_shift (rshift)
- space (as modifier when held)

## RHS Symbol Keys (bare)
- 6 7 8 9 0
- hyphen (-)
- equals (=)
- open_bracket ([)
- close_bracket (])
- backslash (\)
- semicolon (;)
- quote (')
- comma (,)
- period (.)
- slash (/)

## RHS Letter Keys
- y u i o p
- h j k l
- n m

## Thumb Cluster
- page_up
- page_down
- enter
- spacebar
- right_control
- right_command

## Physical Arrows
- up
- down
- left (currently = backspace)
- right (currently = delete)

---

## Current Mappings (Ins Mode)

### right_command + key
| Key | Current |
|-----|---------|
| j | word left |
| k | word right |
| m | line start |
| comma | line end |
| up | left arrow |
| down | right arrow |
| alone | } |
| shift+alone | ] |

### space + key
| Key | Current |
|-----|---------|
| j | shift+word left (select) |
| k | shift+word right (select) |
| m | shift+line start (select) |
| comma | shift+line end (select) |
| up | shift+left (select) |
| down | shift+right (select) |
| enter | cmd+enter |

### quote (')
| Key | Current |
|-----|---------|
| hold | mirror mode |

### page_up / page_down
| Key | Current |
|-----|---------|
| page_down | left click |
| page_up | right click |

---

## Available Slot Combinations

### right_command + (free)
h, l, n, o, p, i, u, y, ;, ., /, [, ], \, 6, 7, 8, 9, 0, -, =

### right_control + (free, but alone exits to Normal)
All RHS keys available as combos

### right_shift + (free)
All RHS keys (would type shifted version normally, could remap)

### space + (free)
h, l, n, o, p, i, u, y, ;, ., /, [, ], \, 6, 7, 8, 9, 0, -, =
