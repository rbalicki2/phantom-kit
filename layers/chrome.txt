Chrome Layer (right_control+J, Chrome only)

H = New tab →Ins, Shift+H = Reopen closed tab
J/K = Prev/Next tab (stays), Ctrl+H = Close (stays), Ctrl+Fn+H = Close+prev (stays)
Fn+J/K = Move tab left/right (stays)
L = Address bar →Ins, Fn+L = Copy URL
N = Cmd+K →Ins
, = Search tabs →Ins
Y/U/I/O = Tab 1/2/3/4
Fn+Y/U/I/O = Tab 5/6/7/8
P = Last tab
M = Dark mode (Alt+D)
. = Refresh, Fn+. = Hard refresh
