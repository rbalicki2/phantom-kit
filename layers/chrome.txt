Chrome Layer (right_control+J, Chrome only)

H = New tab →Ins
J/K = Prev/Next tab (stays), Ctrl+H = Close (stays), Ctrl+Shift+H = Close+prev (stays)
Shift+J/K = Move tab left/right (stays)
L = Address bar →Ins, Shift+L = Copy URL
N = Cmd+K →Ins
, = Search tabs →Ins
Y/U/I/O = Tab 1/2/3/4
Shift+Y/U/I/O = Tab 5/6/7/8
P = Last tab
M = Dark mode (Alt+D)
. = Refresh, Shift+. = Hard refresh
