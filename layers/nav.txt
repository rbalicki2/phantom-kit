Nav Layer (right_control+N)

H = Karabiner-EventViewer, Shift+H = App Switcher
J = iTerm, Shift+J = Window Switcher
K = VS Code, Shift+K = Finder
L = Signal
M = Chrome Personal
, = Chrome Work
N = Spotlight, Shift+N = iTerm tab 1
Shift+M/,/. = iTerm tab 2/3/4
Ctrl+H = Close (Cmd+W)
P = Screenshot full, Ctrl+P = Screenshot selection
U/I = Next/Prev tab
Up/Down = Left/Right half
Enter = Switch windows
Space = Maximize
