Nav Layer (right_control+N)

H = Karabiner-EventViewer, Shift+H = App Switcher
J = iTerm, Shift+J = Window Switcher
K = VS Code, L = Signal, O = Obsidian
M = Chrome Personal, , = Chrome Work
. = Finder, Shift+. = Go to Folder, Ctrl+. = Go to Folder + Paste
Shift+N/M/, = iTerm tab 1/2/3
N = Spotlight, Enter = Switch windows
Ctrl+H = Close (Cmd+W)
P = Screenshot full, Ctrl+P = Screenshot selection
U/I = Prev/Next tab (stays)
Up/Down = Left/Right half, Space = Maximize
