Nav Layer (right_control+N)

H = Karabiner-EventViewer, Shift+H = Chrome Personal tab 1
J = iTerm, Shift+J = Chrome Personal tab 2
K = VS Code, Shift+K = Chrome Work tab 1
L = Signal, Shift+L = Chrome Work tab 2
M = Chrome Personal, Shift+M = iTerm tmux window 2
, = Chrome Work, Shift+, = iTerm tmux window 3
Shift+. = iTerm tmux window 4
Shift+/ = iTerm tmux window 5
N = Spotlight, Shift+N = iTerm tab 2 pane 1
Ctrl+H = Close (Cmd+W)
P = Screenshot full, Ctrl+P = Screenshot selection
U/I = Next/Prev tab
Up/Down = Left/Right half
Enter = Switch windows
Space = Maximize
