INS - Typing Mode
All keys type normally
right_ctrl → Normal
' (hold) → Shift
Shift (hold) → Mirror
Fn+Space → Shift oneshot
Fn+Enter → Shift+Enter

Brackets:
Shift+Up/Down → { }
Quote+Up/Down → [ ]
Quote+,/. → < >
Shift+[ → ', Shift+] → |
Quote+[ → ", Quote+] → ~

Symbols:
Shift+hyphen → _
Quote+hyphen → =
Shift+Quote+hyphen → +
Shift+Quote+Up → \, Shift+Quote+Down → `
Quote+/ → ?
