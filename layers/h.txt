H Layer (right_control+H)

H = plus, Shift+H = Cmd+plus
N = equals, Shift+N = Cmd+equals
Ctrl+H = Close
J/K = Delete word left/right
M/, = Delete to line start/end
Up/Down = Delete char left/right
Shift+above = Select instead of delete

Sub-layers (press, then a-z sends modifier+letter):
Y = Cmd, Ctrl+Y = Ctrl+Cmd
U = Ctrl, Ctrl+U = Ctrl+Alt
I = Alt, Ctrl+I = Alt+Cmd
O = Cmd+Ctrl+Alt
